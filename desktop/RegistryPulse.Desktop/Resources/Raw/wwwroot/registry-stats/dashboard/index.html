Help
Registry Pulse is a local, file-based dashboard for multi-registry package metrics.
Setup: define portfolio, generate snapshot.
Dashboard: view summary, breakdown, leaderboard.
Data: stored locally on Desktop; static JSON on web.
Registries: npm, PyPI, NuGet, VS Code Marketplace, Docker Hub.
Privacy: no telemetry, no accounts, no cloud storage. All data remains on this device.
